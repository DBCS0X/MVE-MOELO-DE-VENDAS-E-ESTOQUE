# Readme 

## O sistema de estoque e vendas foi desenvolvido para auxiliar no controle de produtos e vendas e gerenciamento de um estoque de loja. O sistema permite que o vendedor acompanhe o estado do deu estoque e também permite que o vendedor cadastre produtos, registre venda e acompanhe a disponibilidade de itens para venda. 

 

Funcionalidades  

Cadastro do produto  

Controle de produtos  

Cadastro de produtos  

Edição da descrição do produto  

Exclusão de produto de estoque  

Consultar produtos disponíveis 

 

Controle de vendas  

Registra vendas  

Atualização automática no estoque após a venda  

Histórico de vendas  

Calcular valor total das vendas  

Calcular desfalque do caixa  

 

Relatório do dia 

Produtos mais vendidos  

Quantidade em estoque  

Relatório de vendas pelo período  

Valor total de vendas no dia
